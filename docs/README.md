# Documentation

This directory contains all project documentation organized by category.

## Directory Structure

- **deployment/** - Deployment guides, checklists, and production setup
- **security/** - Security documentation, fixes, and password management
- **setup/** - Initial setup guides and quick start instructions
- **testing/** - Test reports, workflow testing, and testing documentation
- **troubleshooting/** - Troubleshooting guides and fix documentation
- **Root** - General documentation (Frontend README, System Reports)
